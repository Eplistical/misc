# misc

* Some important helper headers

* C++11 is required. (To be specific, GCC 5 is necessary to compile the package!)

To install:
```
 git clone https://github.com/Eplistical/misc.git
 
 cd misc
 
 mkdir build
 
 cd build
 
 cmake -DCMAKE_INSTALL_PREFIX=<install_prefix> .. 
 
 make
 
 make install
 
 make test
```
